I developed working on Greetings 101, a language-learning web app designed to promote inclusivity and reduce stigmas around multilingualism. The frontend of the app uses Next.js and Typescript. The backend was developed with Django.

Some of the features that have been implemented are: An interactive Spanish module, user authentication, analytics, profile customization, and a flashcard practice game. I plan to add much more in the future, but my next steps will be refining the Spanish 101 module and adding to the ASL 101 module.
