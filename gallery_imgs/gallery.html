CSS Animation: Trains

Explore the benefits of public tranpotation while CSS animations craft a visually compelling experience. The GIFs, sourced from the web, dictated the project's color scheme, influincing vibrancy and futuristic themes. Notable features include a hover-triggered animated pie chart that visually represents key data and a timed animation at the bottom (reload page to see), offering an engaging user experience. This project not only demonstrates the power of CSS animations, but also serves as a platform to promote public transportation through an aesthetically pleasing and interactive lens.
